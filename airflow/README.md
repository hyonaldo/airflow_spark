# airflow template
This template helps you initialize airflow and set spark running environments

## requirements
* python >= 3 

## install & init airflow

```
$ cd airflow
$ ./init-airflow.sh
```

## start airflow
```
$ cd airflow
$ ./start-airflow.sh
```

## test
```
$ cd airflow/dags
$ python3 template.py
```
